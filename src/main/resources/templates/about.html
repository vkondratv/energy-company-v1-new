О проекте

Информационно-справочная система для энергетической компании разработана с использованием современных технологий:

  • Spring Boot 2.7+
  • Spring Security для аутентификации и авторизации
  • Spring Data JPA для работы с базой данных
  • Thymeleaf для отображения представлений
  • Bootstrap 5 для адаптивного дизайна

Система обеспечивает безопасное хранение и управление данными энергетических объектов с разграничением прав доступа.
