О проекте

Система управления энергообъектами - это веб-приложение для учета и контроля энергетических объектов различных типов (АЭС, ГЭС, ТЭС, ТЭЦ, СЭС, ВЭС).

Основные возможности системы:

  • Управление энергообъектами (CRUD операции)
  • Фильтрация и поиск объектов
  • Статистика и аналитика
  • Безопасная авторизация пользователей
  • Адаптивный пользовательский интерфейс
Цели проекта

Разработка учебного проекта для демонстрации навыков работы с современными веб-технологиями и фреймворками.

Проект реализован в рамках учебной программы для получения практического опыта разработки полнофункционального веб-приложения.
